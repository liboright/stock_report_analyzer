## 主线（示例片段）

> 一句话定位：全球动力电池龙头，连续 9 年市占率第一，2025 年储能业务从配套跃升为第二增长曲线。
>
> 核心故事：动力电池主业稳健（+17% 营收 / +39% 销量）→ 储能业务规模化（占比从 5% 升至 15%+）→ 全球化与神行超充技术领先 → 风险来自原材料价格与海外政策。
>
> 横向贯穿主题：
> 1. 储能第二曲线（业务第三节 + 财务第五节 + 趋势第六节 + 展望第九节）
> 2. 全球化战略（业务第三节 + 财务第五节 + 第七节）
> 3. 神行超充技术（第七节 + 未来展望）

## 报告 9 节速览

1. **执行摘要** — 600-800 字，4 段叙述 + 1 张关键指标表
2. **公司画像** — 300-400 字 + 1 张基本信息表
3. **主要业务** — 800-1200 字（动力电池 / 储能电池 / 电池材料 / 电池矿资源）+ 2 张表
4. **新业务** — 400-600 字（储能业务 + 换电服务 + 飞机/船舶电动化）+ 1 张表
5. **盈利模式** — 600-800 字 + 2 张表（收入构成 + 盈利质量）
6. **发展现状与趋势** — 500-700 字 + 1 张行业地位演变表
7. **最近一期财务深度解读** — 500-700 字 + 1 张关键变化表（**差异化亮点**）
8. **风险提示** — 300-400 字
9. **未来展望** — 400-500 字

## 备注

- 本示例数据来自宁德时代 2023-2025 年报 stage1 实际产物
- v1 初稿（write mode）不强调措辞，专注覆盖
- 主线文档（mainline mode）提炼贯穿 9 节的"读者读完后应该带走的故事"
- v2 重写（rewrite mode）以主线为骨架，强制去重 + 措辞润色
- 审查（review mode）用 general-purpose SubAgent + JSON prompt
- 润色循环默认 max_polish_rounds=2，本例在 v3 通过审查（共 1 轮润色）
- 中间稿和审查报告保留在 `_drafts/`，可清理
