## 一、公司基本定位

用 **200-400 字**叙述性段落回答「这是什么公司、做什么的」：
- **公司性质与定位**：用一句话点明公司是谁、行业地位（如"全球 XX 行业龙头"、"国内 XX 细分赛道头部企业"）。
- **业务范围**：覆盖哪些业务板块（按公司实际业务描述，如产品制造、服务提供、资源开发等）。
- **上市信息**：A 股 / H 股 / 美股代码及主要上市地（如适用）。
- **注册地、总部、员工规模等**基本信息（如年报披露）。
- **整体规模感**：用 1-2 个核心数据让读者快速形成规模认知（如"年营收超 X 亿元、核心产品年销量超 X 单位"）。

写作要点：用讲故事的语气把公司介绍清楚，避免"清单式"罗列。

## 二、业务板块拆解

**按营收占比从大到小**，每个板块用 **150-300 字**叙述 + **1 张「业务数据 + 收入毛利率」组合表**（2023-2025 三年），回答「这是干什么的、赚多少钱、定位如何」：
- **板块名称 + 2025 年营收占比**（如「XX 业务（74.7% 营收）」）
- **产品类型与目标客户**：主要做什么？卖给谁？什么场景用？
- **2025 年关键表现**：销量、出货量、收入、同比变化、毛利率。
- **市场地位**：全球第几？连续几年？市占率多少？

**强制要求**：
- 至少覆盖**营收占比前 4 名板块**（按 2025 年口径；若公司业务简单可适当精简）。
- 板块排序按营收占比**从大到小**，不要按"重要性"主观排序。
- 每个板块结尾**必须**有数据支撑（至少一个销量/收入/市占率数字）。
- **板块命名与拆解方式根据公司实际业务调整**，板块名称使用年报中的原文表述。

**每个板块的「业务数据 + 收入毛利率」组合表**（2023-2025 三年口径）：

| 指标 | 2023 年 | 2024 年 | 2025 年 |
|------|---------|---------|---------|
| 销量（万单位/万套/GWh，按公司实际口径） | | | |
| 销量同比 | | | |
| 收入（亿元） | | | |
| 收入同比 | | | |
| 营收占比 | | | |
| 毛利率（%） | | | |

> **数据要求**：
> - 销量单位必须按公司年报中的实际口径（电池用 GWh、消费品用万套/万吨、制造用万台等）。
> - 若年报仅披露收入未披露销量（如材料、矿产、电池回收等）→ 销量行留空，**不要编造**。
> - 营收占比 = 该板块收入 / 公司总营收。
> - 毛利率保留两位小数（如 23.84%）。
> - **如某年份未披露某指标**，在单元格内写 "未披露"，**不要留空**也不要外推。

## 三、主要产品矩阵

用 **200-400 字**叙述介绍公司的产品线，避免逐个产品清单式罗列：
- **产品类别划分**：按公司的实际分类方式（如按技术路线、按应用场景、按客户类型等）。
- **代表产品系列**：可点名 3-5 个旗舰产品，用 **1 张表**展示（产品名 + 类别 + 核心卖点 + 目标场景）。
- **新产品/新技术**：本年推出的代表性新品，点出差异化亮点。

**写作要点**：
- 用产品组合的方式让读者理解公司的产品布局广度与深度。
- 所有具体产品名/技术名/客户名由 report_writer 根据公司年报实际数据填充，不要预设。

## 四、经营业绩（近三年趋势）

用 **300-400 字**叙述 + **1 张关键数据表** + **3 张分类汇总表**（分业务/分产品/分地区收入与毛利率）回答「近三年公司整体表现如何演变」：
- **整体趋势**：以 2023-2025 三年为时间轴，叙述营收、净利润、现金流等核心指标的同比变化趋势，分析增长动能从"加速→减速→承压"或"平稳→加速→突破"等不同模式。
- **盈利能力变化**：毛利率、净利率、ROE 等盈利质量指标的演变，识别量增、结构升级、降本增效等驱动因素。
- **关键经营事件**：报告期内发生的重大事件（上市、回购、分红、重大订单、产能扩张、并购重组等）。
- **运营效率指标**：产能利用率、产销率、人均产出等（如年报披露）。

| 产品类别 | 2023 年收入 | 2023 占比 | 2023 毛利率 | 2024 年收入 | 2024 占比 | 2024 毛利率 | 2025 年收入 | 2025 占比 | 2025 毛利率 |

## 六、主要客户与销售情况

用 **150-250 字**叙述 + **1 张客户表**回答「客户是谁、是否过度依赖、卖到哪」：
- **客户结构**：覆盖哪些类型客户，是否分散。
- **前五大客户占比**：体现客户集中度风险（如果高度集中需特别说明）。
- **地域分布**：境内 vs 境外收入占比及毛利率（如显著差异需重点说明）。
- **销售渠道**：直销/经销、线上/线下、海外网络等。

**表格示例**：前五大客户销售额及占比（公司名按年报披露，未披露则用"第一名/第二名"代称）。

## 七、核心竞争优势（护城河）

用 **300-500 字**叙述公司**靠什么保持领先**，分 2-4 个维度展开：
- **规模与制造壁垒**：产能规模、产能利用率、规模效应摊薄成本。
- **技术研发壁垒**：研发投入、研发人员、专利数量、覆盖技术路线的广度。
- **客户与渠道壁垒**：客户分散度、海外渠道、品牌粘性等。
- **资源 / 产业链壁垒**：上游资源掌控、产业链垂直整合、生态闭环等。

**写作要点**：
- 每个维度必须**有数据支撑**（不是空洞表述）。
- 不要写成营销文，要让读者看到"具体强在哪、强多少"。
- 数据来源于年报「核心竞争力」章节。
- **根据公司实际优势维度调整**，有的公司是品牌（如消费品）、有的是资源（如矿产）、有的是规模（如制造）。

## 八、未来增长逻辑

用 **300-500 字**叙述公司**未来靠什么持续增长**，分 3-5 个核心驱动展开：
- **赛道层面**：公司所在赛道的中长期景气度与增量空间。
- **产品 / 技术层面**：新技术、新产品的落地节奏与商业化预期。
- **场景层面**：新场景（新应用领域、新客户群）的拓展。
- **全球化层面**：海外市场扩张节奏与本地化能力。
- **效率 / 成本层面**：降本增效、规模效应释放、运营效率提升等。

**写作要点**：
- 每个增长点要回答「为什么是现在」「公司凭什么抓住」。
- 避免空泛愿景，必须有年报中披露的具体进展作为支撑。
- **根据公司行业属性调整**（如消费品的增长看品牌力/渠道下沉/新品类；制造业的增长看产能/技术/全球化）。

## 九、风险提示

用 **200-400 字**叙述公司**面临的核心风险**，分 3-5 个风险点：
- **行业层面**：行业产能过剩、价格战、行业增速放缓、政策退坡等。
- **需求层面**：下游需求疲软、客户预算收缩、替代品冲击等。
- **成本 / 资源层面**：原材料价格波动、能源价格、汇率波动等。
- **竞争 / 技术层面**：技术路线被颠覆、竞争对手追赶、客户流失等。
- **公司层面**：客户集中度、应收账款、商誉减值、合规风险等特定风险（如年报披露）。

**写作要点**：
- 每个风险要说明「为什么是风险」「影响多大」。
- 数据来源于年报「未来发展展望」中披露的风险因素。
- **根据公司实际风险调整**，不同行业风险类型差异很大（互联网看监管/数据安全；制造业看产能/原材料；消费品看品牌/渠道）。

## 写作硬约束

1. **叙述优先**：每节以段落叙述为主，**单节最多 1-2 张表格**（第四节含 3 张汇总表除外）。整篇文档表格总数**不超过 12 张**（含 4 张业务板块表 + 3 张分类汇总表 + 5 张其他表）。
2. **段落长度**：单段建议 100-300 字，避免一个数据一行。
3. **数据点睛**：表格用于「分业务经营数据」「分业务/分产品/分地区收入与毛利率」「三年财务对比」「客户排名」「市场地位演变」等关键对比场景，不要用表格代替叙述。
4. **读者画像**：默认读者不熟悉该公司，目标是让他们读完后能向他人复述「这家公司是干什么的、做得怎么样、靠什么领先、未来靠什么增长、有什么风险」。
5. **不输出**：**不输出投资建议、估值判断、目标价、买卖建议**。本 skill 只做"业务和行业理解"。
6. **数据来源**：所有数据必须来自年报正文，不要网络搜索补充。
7. **由近及远**：以 2025 年为基准，2024/2023 用于趋势对比。
8. **行业通用**：模板不绑定任何特定行业（电池/白酒/医药/互联网等），所有字段、表述、产品类别均**根据公司实际数据填充**。
