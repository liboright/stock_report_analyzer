# 公司业务概况

## 一、公司基本信息
- 公司名称（全称）：宁德时代新能源科技股份有限公司
- 股票代码：300750
- 上市地点：深圳证券交易所创业板
- 注册地址：福建省宁德市蕉城区漳湾镇金湾路1号
- 主营业务（一句话概括）：全球领先的零碳新能源科技公司，主要从事动力电池、储能电池的研发、生产、销售及服务。

## 二、主营业务详解
- 公司所处产业链环节：
  公司处于锂离子电池产业链中游，专注于电池研发、生产、销售及系统集成，同时向上游延伸至电池材料、回收及矿产资源领域。

- 业务情况（收入、毛利率、市场份额）：
  公司为全球领先的动力电池和储能电池供应商。根据SNE Research数据，2025年全球动力电池使用量达1,187GWh，同比增长31.7%；全球储能电池出货量550GWh，同比增长79%。公司在动力电池和储能电池领域的市场份额持续保持全球领先。

## 三、主要产品/服务

### （一）动力电池系统
- 产品名称：动力电池系统（包括电芯、模组/电箱及电池包）
- 化学体系：磷酸铁锂电池、三元高压中镍电池、三元高镍电池、超混电池、钠离子电池、凝聚态电池等
- 应用领域：乘用车（BEV、REV、PHEV、HEV）、商用车（客车、城市配送、重载运输）、船舶、航空器、电动工具、电动两轮车等

### （二）储能电池系统
- 产品名称：储能电池系统（包括电芯、电池柜、储能集装箱及系统集成）
- 代表产品：EnerOne、EnerOnePlus、EnerC、EnerC Plus、EnerD、EnerX、天恒储能系统、TENER Stack等
- 应用领域：表前储能（公用事业储能）、表后储能（工商业储能、家庭储能、数据中心储能）

### （三）电池材料、回收及矿产资源
- 产品名称：锂盐、前驱体及正极材料
- 业务模式：通过回收废旧电池提取镍、钴、锰、锂、磷等金属材料，生产电池原材料；同时通过自建、参股、合资等方式布局电池矿产资源

### 主要客户情况
- 公司客户覆盖全球主流车企、储能系统集成商等，详情请参阅公司年报披露信息。
