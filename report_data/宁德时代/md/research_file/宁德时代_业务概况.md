## 二、主要产品及服务

### 2.1 动力电池系统

公司动力电池产品包括电芯、模组/电箱及电池包。可提供：
- 磷酸铁锂电池
- 三元高压中镍电池
- 三元高镍电池
- 超混电池
- 钠离子电池
- 凝聚态电池

**应用领域**：
- 乘用车：BEV、REV、PHEV、HEV
- 商用车：道路客运、城市配送、重载运输、道路清洁
- 新兴领域：船舶、航空器、电动工具、电动两轮车

**2025年新技术**：
- 二代神行超充电池：全球首款兼具800公里续航和峰值12C超充速度的磷酸铁锂电池
- 骁遥双核电池：跨化学体系设计，综合性能全面提升
- 钠新乘用车动力电池：优异低温性能，降低对锂资源依赖

### 2.2 储能电池系统

公司提供电芯、电池柜、储能集装箱以及系统集成等储能解决方案。

**表前领域**：
- EnerOne、EnerOnePlus 户外液冷电池柜
- EnerC、EnerC Plus、EnerD、EnerX 集装箱式液冷电池柜
- 天恒储能系统（6.25MWh）
- TENER Stack（9MWh超大容量储能系统）

**表后领域**：
- PR系列、Unic系列、安鑫系列、PU系列

### 2.3 电池材料、回收及矿产资源

- 锂盐、前驱体及正极材料
- 废旧电池回收：镍、钴、锰、锂、磷、铁、铝、铜回收
- 矿产资源：锂、镍、钴、磷矿权投资

### 2.4 新兴领域创新解决方案

- **巧克力换电**：乘用车换电网络，超1,000座换电站
- **骐骥换电**：重卡换电，超300座换电站
- **eVTOL**：吨级eVTOL航空器（凯瑞鸥）
- **电动船舶**：兆瓦级充电、分钟级换电

## 四、经营模式

| 环节 | 模式 |
|------|------|
| 研发 | 以自主研发为主、外部合作为辅，数字化智能化贯穿 |

| 生产 | 自建生产基地为主，合资建厂、技术授权为辅 |
| 销售 | 全球客户直销+换电服务网络 |
