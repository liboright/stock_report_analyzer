# 行业分析

## 一、行业政策

- **国家政策支持：**
  - 2025年1月，国家发展改革委、财政部发布《关于2025年加力扩围实施大规模设备更新和消费品以旧换新政策的通知》，推动老旧营运货车报废更新，提高新能源城市公交车及动力电池更新补贴标准，扩大汽车报废更新支持范围。
  - 2025年1月，国家发展改革委、国家能源局发布《关于深化新能源上网电价市场化改革促进新能源高质量发展的通知》，取消"强制配储"政策，推动新能源上网电量全面进入电力市场。
  - 2025年9月，国家发展改革委、国家能源局发布《新型储能规模化建设专项行动方案(2025-2027年)》，提出到2027年全国新型储能装机规模达到1.8亿千瓦以上，带动直接投资约2,500亿元。
  - 2025年12月，国家发展改革委、工业和信息化部、国家能源局发布《关于印发国家级零碳园区建设名单(第一批)的通知》，推动零碳园区建设示范引领。
  - 海外方面，欧盟委员会2025年3月发布《欧洲汽车行业工业行动计划》，通过"社会租赁计划"、"企业车队法规"等举措促进电动车渗透率提升，并提供最高30亿欧元支持本土电池制造能力建设。

- **监管环境：**
  - 2025年6月，生态环境部等七部委发布《关于规范锂离子电池用再生黑粉原料、再生钢铁原料进口管理有关事项的公告》，对锂离子电池用再生黑粉原料实行规范管理。
  - 2025年12月，工业和信息化部发布《关于开展汽车动力电池碳足迹申报工作的通知》，推动动力电池碳足迹核算规则建立健全。

## 二、市场空间

- **市场规模：**
  - 全球新能源车市场：根据SNE Research数据，2025年全球新能源车销量为2,147.0万辆，同比增长21.5%。
  - 国内新能源车市场：根据中国汽车工业协会数据，2025年中国新能源车销量为1,387.5万辆，其中新能源乘用车销量为1,300.5万辆，同比增长17.7%，渗透率提升至54.0%；新能源商用车销量为87.1万辆，同比增长63.7%，渗透率提升至26.9%。
  - 欧洲新能源车市场：根据欧洲汽车制造商协会数据，2025年欧洲新能源乘用车销量为385.8万辆，同比增长30.9%，渗透率达29.1%。
  - 动力电池市场：根据SNE Research数据，2025年全球动力电池使用量为1,187GWh，同比增长31.7%。
  - 储能市场：根据SNE Research统计，2025年全球储能电池出货量550GWh，同比增长79%。根据中关村储能产业技术联盟统计，2025年我国新型储能新增装机规模达189.5GWh，同比增长73%。

- **增长趋势：**
  - 新能源车渗透率持续提升，国内新能源乘用车渗透率已超50%，进入全面市场化阶段。
  - 储能市场爆发式增长，受益于电力需求增长、清洁能源快速部署及储能成本下降。
  - 2025年我国新增并网风电光伏装机容量438GW，同比增长22.3%，光伏、风电累计装机容量首次超过煤电装机，新能源正从补充能源加速向主体能源迈进。

## 三、产业链情况

锂电池产业链涵盖上游原材料、中游制造及下游应用三大环节：

- **上游原材料**：主要包括锂、钴、镍等矿产资源，以及正极材料（磷酸铁锂、三元材料）、负极材料、隔膜、电解液等电池材料。根据SMM统计，2025年我国三元与磷酸铁锂正极材料合计产量达456.5万吨，同比增长51%。随着早期投放市场的锂电池逐渐进入退役期，退役电池回收需求逐步提升，2025年我国锂电池报废量达81.9万吨，同比增长9%。

- **中游制造**：涵盖动力电池和储能电池的生产制造。全球动力电池使用量1,187GWh，储能电池出货量550GWh（数据来源：2025年年报）。

- **下游应用**：主要应用于新能源汽车及储能系统。新能源汽车领域，电动化、智能化趋势加速；储能领域，数据中心能源需求及灵活性资源调节需求增加，峰谷价差拉大提升项目经济性（数据来源：2025年年报）。

## 四、发展趋势

- **技术路线：**
  - 电动化、智能化深度融合：电力和交通行业迎来能源转型的深刻变革，智能化、电动化趋势下，新能源汽车对动力电池性能要求持续提升。
  - 高品质锂电池成为核心载体：高品质锂电池凭借高能量密度、长循环寿命、良好稳定性及安全性等性能优势，在新型能源系统中成为支撑能源系统缓冲、稳定与调度的关键单元。
  - 构网型储能技术推进：在高比例新能源电网、弱电网和孤岛电网中应用。

- **产品升级方向：**
  - 单车带电量持续提升，带动动力电池需求增长。
  - 储能系统向大规模、智能化方向发展，探索"人工智能+"应用场景。
  - 绿电直连、虚拟电厂、智能微电网、源网荷储一体化、车网互动等新模式不断涌现。
  - 电池材料向高能量密度、高安全性、低成本方向升级。

## 五、市场格局

- **主要玩家：**
  - 宁德时代是全球动力电池和储能电池领域的绝对领先企业。

- **市场份额：**
  - 动力电池领域：2025年公司动力电池使用量全球市占率为39.2%，较去年同期提升1.2个百分点，已连续9年（2017-2025年）排名全球第一。
  - 储能领域：公司已连续5年（2021-2025年）储能电池出货量排名全球第一。

- **竞争态势：**
  - 全球竞争格局相对集中，宁德时代凭借技术优势、规模优势及全球化布局，持续巩固市场领导地位。
  - 在全球碳中和目标推动下，动力电池和储能电池市场需求持续增长，行业进入门槛提高，技术领先企业优势进一步凸显。
  - 欧洲、北美等地区持续推进本土化供应链建设，但中国企业在技术积累、产能规模及成本控制方面仍具显著优势。
