# 公司业务概况

## 一、公司基本信息
- 公司名称（全称）：宁德时代新能源科技股份有限公司（CATL）
- 股票代码：300750（A股）、03750（H股）
- 上市地点：深圳证券交易所（A股）、香港联合交易所有限公司（H股）
- 注册地址：福建省宁德市蕉城区漳湾镇新港路2号
- 主营业务（一句话概括）：全球领先的零碳新能源科技公司，主要从事动力电池、储能电池的研发、生产、销售。

## 二、主营业务详解
- 公司所处产业链环节：
  - 锂电池产业链中游核心环节（电池制造）
  - 上游：电池材料、矿产资源
  - 下游：新能源汽车、储能系统、新兴应用领域

- 业务情况（收入、毛利率、市场份额）：
  - 动力电池：全球市占率连续9年第一（2025年39.2%）
  - 储能电池：全球出货量连续5年第一
  - 全球布局：六大研发中心、24家电池工厂

## 三、主要产品/服务

### 3.1 动力电池系统
- 产品类型：电芯、模组/电箱及电池包
- 化学体系：磷酸铁锂电池、三元高压中镍电池、三元高镍电池、超混电池、钠离子电池、凝聚态电池
- 应用领域：乘用车（BEV、REV、PHEV、HEV）、商用车、船舶、航空器、电动工具、电动两轮车

### 3.2 储能电池系统
- 产品类型：电芯、电池柜、储能集装箱、系统集成
- 表前储能：EnerOne、EnerOnePlus、EnerC、EnerC Plus、EnerD、EnerX、天恒储能系统（6.25MWh）、TENER Stack（9MWh）
- 表后储能：PR系列、Unic系列、安鑫系列、PU系列

### 3.3 电池材料、回收及矿产资源
- 电池材料：锂盐、前驱体及正极材料
- 回收业务：废旧电池中镍、钴、锰、锂、磷、铁、铝、铜等金属材料回收再利用
- 矿产资源：锂、镍、钴、磷矿权投资

### 3.4 新兴创新解决方案
- 巧克力换电（乘用车）：超1,000座换电站
- 骐骥换电（重卡）：超300座换电站
- eVTOL航空器：吨级eVTOL（凯瑞鸥）
- 电动船舶：兆瓦级充电、分钟级换电
